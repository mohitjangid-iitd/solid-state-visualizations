# Solid-State Visualizations

This repository contains VPython-based 3D visualizations of crystal structures, first Brillouin zones, and geometric constructions used in Solid-State Physics coursework.

## Contents
- `solid_state_assignment_1_v2.py` — Simple lattice visualization
- `solid_state_assignment_2_v2.py` — Tetrahedral and ice-like models
- `solid_state_assignment_3_v2.py` — Brillouin zone and reciprocal lattice
- `FCC.py` — Face-centered cubic visualization

## Tools Used
- VPython
- NumPy
